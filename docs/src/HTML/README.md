## 渐进增强和优雅降级

- `渐进增强`：针对低版本浏览器进行构建页面，保证最基本的功能，然后再针对高级浏览器进行效果、交互等改进和追加功能，达到更好的用户体验。
- `优雅降级`：一开始就构建完整的功能，然后再针对低版本的浏览器进行兼容。
